Проект пишеться с целью сдачи курсового проекта, а так же для проверки собственных знаний на практике. 
Тема проекта: Умный Холодильник 
В качестве графического интерфеса используется консольная псевдографика.
Базовый функционал:
-Добавление/удаления продуктов 
-Просмотр и сортировка по параметрам
-Вывод истории в файл
-Поиск нужного продукта
-Автоматический список покупок
Остальной функционал дописывается исходя из желания добавить новые функции и проверить новые знания на практике.
